# Contributing

Thank you for your interest in contributing to `skill-autonomous-orchestrator`.

## Development setup

```bash
git clone https://github.com/metyatech/skill-autonomous-orchestrator.git
cd skill-autonomous-orchestrator
```

## Submitting changes

1. Fork the repository and create a feature branch.
2. Add or update tests for any changed behavior.
3. Open a pull request with a clear description of the change.

## Scope

This skill covers **autonomous meta-orchestrator procedures**. Please keep PRs scoped to multi-agent coordination and lifecycle management.
